## Development

- Read docs/overview.md to know the basic of this project
- Follow recommendations for Android architecture:
    - Model-View-ViewModel
    - UI Layer & Data Layer, but no domain layer
    - Unidirectional data flow
    - Decouple with Dependency Injection
- Follow Material Design 3 guideline to design composable element
- Generate icons in res/drawable/ instead of using Compose Material Icons Extended
